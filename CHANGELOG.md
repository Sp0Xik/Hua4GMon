# Changelog

Все значимые изменения в этом проекте документируются здесь.
Формат: [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
версионирование: [SemVer](https://semver.org/lang/ru/).

## [1.2.0] — 2026-XX-XX

### Добавлено
- Индикатор тенденции RSRP (↑ / → / ↓) с подсказкой для монтажника.
- Авто-переподключение к роутеру при обрыве с экспоненциальным backoff.
- Экспорт сессии измерений в CSV для отчётов клиенту.
- CLI-аргументы `--ip`, `--password`, `--verbose`, `--version`
  для быстрого запуска без GUI-ввода.
- Расшифровка LTE-бандов из любых форматов API (`'7'`, `'LTE BAND 20'`,
  `'B3+B7'`, битмаски `0x40`) в человекочитаемый вид с указанием частоты.
- Поле EARFCN на вкладке «Вышка» + fallback-определение band по EARFCN.
- Блок «SIM / Устройство» на вкладке «Вышка»: IMEI, IMSI, ICCID, номер
  телефона, серийный номер, модель, версия прошивки.
- Кнопка «🔄 Перезагрузить роутер» на вкладке «Сеть».
- Вкладка «🛡 Белые списки (РФ)»: проверка через TCP-сокет на `:443`
  для группы заведомо разрешённых сайтов (Госуслуги, Сбер, mos.ru) и
  контрольных нейтральных. Определяет, включён ли на БС фильтр.
- Чекбокс «Поверх окон» для работы во время монтажа.
- Аудио-помощник: частота тона привязана к разнице RSRP от пика сессии.
- Расширенный список операторов (PLMN_MAP) для РФ, РБ, KZ, UA.
- Кнопка «Сбросить пики».
- Тесты на pytest (`tests/test_pure.py`) — 76 кейсов чистых функций.
- Линтер ruff и CI-проверки lint + test + build.
- GitHub Actions: автоматический Release на тегах `vX.Y.Z`, с zip-архивом,
  SHA256 и заметками к релизу.

### Изменено
- График полностью переписан с matplotlib на голый `tk.Canvas`
  (класс `CanvasGraph`). Размер .exe уменьшается на ~30 МБ, запуск быстрее.
- Переключатель Standard/Pro убран — все элементы интерфейса теперь
  доступны одновременно. Объединённый экран компактнее и понятнее.
- Парсинг `cell_id` теперь отбрасывает заведомо невалидные значения
  (`0xFFFFFFFF`, > 28 бит).
- `extract_number` использует строгий `re.fullmatch` — больше не
  превращает строки вроде `"timeout 0"` в число `0`.

### Исправлено
- Гонка потоков: фоновый поток мониторинга больше не падает, когда
  основной поток обнуляет `self.client` во время `disconnect()`.
- Корректный `logout()` при отключении.
- Все «голые» `except: pass` заменены на конкретные ветки + `logging`.
- `apply_antenna` теперь устойчив к локализованным меткам.
- `open_cellmapper` валидирует PLMN и eNodeB перед открытием URL.
- `reset_peaks` теперь обновляет UI-лейблы.

### Удалено
- Сохранение пароля и IP в `config.ini` (включая base64-обфускацию).
  Программа стала полностью portable: ничего не пишет на диск кроме
  CSV-экспорта по явному запросу.
- Зависимость от `matplotlib` (~30 МБ из .exe).
- Зависимость от `requests` — вместе с интеграцией InfoCellTowers,
  чей API нестабилен и требует подписки.
- Кнопка «Инфо из InfoCellTowers» с вкладки «Вышка» (CellMapper
  остался — он не требует API-ключа).
