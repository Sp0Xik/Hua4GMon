# Hua4GMon — монитор LTE для роутеров Huawei

![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20Android-blue)
![Version](https://img.shields.io/badge/version-1.3.0-brightgreen)
![Python](https://img.shields.io/badge/python-3.10%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Утилита для мониторинга 4G/LTE-роутеров и модемов Huawei (E3372, B315,
B525, B535, B628, B818 и совместимых). Сделана под задачи **монтажников
антенн**: помогает в реальном времени видеть, как меняется сигнал при
повороте антенны, выбрать лучшую базовую станцию и зафиксировать
результат отчётом клиенту.

Один репозиторий — **две платформы**:

| Платформа | Технологии | Точка входа |
| :-------- | :--------- | :---------- |
| Windows (десктоп) | Tkinter + PyInstaller | `main.py` |
| Android (телефон/планшет) | Kivy + Buildozer | `android_main.py` |

Вся прикладная логика (разбор сигнала, бандов, PLMN, оценка качества,
переводы) вынесена в общий пакет `core/` и переиспользуется обеими
версиями.

## Возможности

- **Мониторинг параметров LTE:** RSRP, RSSI, RSRQ, SINR, скорости DL/UL,
  трафик за сессию, температура чипа, время сессии.
- **Индикатор тенденции** (↑ / → / ↓) — главный инструмент монтажа:
  показывает, улучшается ли сигнал при повороте антенны.
- **Графики** параметров в реальном времени (на Android — с
  полноэкранным режимом и выбором параметра).
- **Информация о вышке:** оператор по PLMN, рабочий LTE-Band с частотой,
  EARFCN, агрегация (CA), ширина канала, eNodeB, локальный сектор, PCI.
- **Информация о SIM/устройстве:** IMEI, IMSI, ICCID, номер телефона,
  серийный номер, модель, версия прошивки.
- **Band Lock и переключение антенн** (внутренняя/внешняя/смешанная)
  через API роутера.
- **Перезагрузка роутера** одной кнопкой (полезно после Band Lock).
- **Тест «белых списков» (РФ):** определяет, включён ли на БС режим, при
  котором работают только сайты из перечня.
- **Авто-переподключение** при обрыве связи с роутером.
- **Два языка интерфейса:** русский и английский.
- **Тестовый режим** (Android) — демо-данные без модема, для проверки на
  эмуляторе.

Только на Windows: «крышный режим» (полноэкранные крупные цифры),
аудио-помощник, экспорт сессии в CSV.

## Установка и запуск

### Windows

Скачайте `Hua4GMon.exe` из [Releases], положите в любую папку и
запустите. Установка не требуется, на диск ничего не пишется (кроме
CSV-экспорта по явному запросу).

[Releases]: https://github.com/Sp0Xik/Hua4GMon/releases

При первом запуске неподписанного .exe Windows SmartScreen может
предупредить «Неизвестный издатель» — это нормально, нажмите
«Подробнее» → «Выполнить в любом случае».

### Android

Скачайте `Hua4GMon-*.apk` из [Releases] и установите (потребуется
разрешить установку из неизвестных источников). Поддерживается Android
7.0+ (API 24).

### Из исходников (Windows / любая ОС с Tk)

```bash
git clone https://github.com/Sp0Xik/Hua4GMon
cd Hua4GMon
pip install -r requirements.txt
python main.py
```

CLI-флаги: `--ip 192.168.1.1 --password admin` (автоподключение),
`--verbose`, `--version`.

## Использование

1. Введите IP роутера (по умолчанию `192.168.8.1`, для B315/B525 —
   `192.168.1.1`) и пароль (со стикера), нажмите «Подключиться».
   Кнопка/раздел «Подсказка» поясняет частые ошибки (401, 108003 и др.).
2. На экране монитора следите за крупными значениями RSRP/SINR/RSSI/RSRQ
   с пиками, стрелкой тенденции и графиком. Двигайте антенну,
   ориентируясь на стрелку (↑ = лучше).
3. Раздел «Вышка»/«Инфо» показывает, к какой БС вы подключены, и данные
   SIM/устройства.
4. Найдя лучшее положение, зафиксируйте частоту в «Сеть» → Band Lock
   и/или переключите антенну на внешнюю. Там же — перезагрузка роутера.
5. «Белые списки (РФ)» — быстрая диагностика, включён ли на БС режим
   фильтрации.

## Поддерживаемые устройства

Hua4GMon работает поверх библиотеки
[huawei-lte-api](https://github.com/Salamek/huawei-lte-api), поэтому
совместим с теми же устройствами. Роутер должен поддерживать сетевой
режим (HiLink), а не serial-режим.

Список устройств, протестированных авторами huawei-lte-api:

```
B310s-22, B311-221, B315s-22, B525s-23a, B525s-65a, B715s-23c, B528s,
B535-232, B628-265, B612-233, B818-263, E5180s-22, E5186s-22a,
E5576-320, E5577Cs-321, E8231, E5573s-320, SoyeaLink B535-333, E3131,
E8372h-608, E3372, E3531, E5530As-2, 5G CPE Pro 2 (H122-373),
5G CPE Pro (H112-372), 5G CPE6 (H165-383)
```

С большой вероятностью работают и другие LTE-устройства Huawei.
Актуальный список — в [репозитории huawei-lte-api][hla].

[hla]: https://github.com/Salamek/huawei-lte-api#tested-on

## Сборка из исходников

**Windows .exe:**
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --noupx --name Hua4GMon main.py
```

**Android .apk:**
```bash
pip install buildozer
# android_main.py копируется как main.py, см. .github/workflows/build-android.yml
buildozer android debug
```

CI на GitHub Actions собирает обе платформы автоматически на push тега
`vX.Y.Z` — см. `.github/workflows/`.
